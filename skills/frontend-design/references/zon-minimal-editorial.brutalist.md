# Zon Minimal Editorial — Brutalist

Use this when the user wants a sharper, more “raw / poster-like” editorial: hard borders, no blur, high contrast, utilitarian rhythm.

## Visual Direction

- Pure black/white, thick rules, squared corners.
- Labels: monospace + aggressive tracking.
- Remove glass/blur; keep layout strict.

## Best For

- Manifestos, punchy outlines, ops checklists, “hard truth” notes.
