# Zon Minimal Editorial — Typewriter

Use this when the user wants a “typed manuscript” feel while keeping the same editorial layout.

## Typography

- Body font: monospace stack (`ui-monospace`, SFMono, Menlo…)
- Slightly tighter sizes: body `15–16px`, answers `16–17px`

## Visual Direction

- Paper texture: subtle horizontal lines or faint noise (keep extremely light).
- Borders stay crisp; header can be non-blurred to feel “printed”.

## Best For

- Draft scripts, transcripts, working notes, research logs.
