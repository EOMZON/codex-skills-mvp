# Zon Minimal Editorial — Luxury Serif

Use this when the user wants a more “book / magazine feature” vibe: serif body, calmer rhythm, slightly more premium.

## Typography

- Body: serif stack (`Iowan Old Style`, `Palatino`, `Songti SC`, `Noto Serif CJK SC`)
- Labels: keep sans/mono-ish small caps feel via tracking + uppercase

## Palette

- Still near-monochrome; allow a muted metallic accent (`--accent: #b59a6a`) for rules/blockquote border.

## Best For

- Profiles, feature essays, portfolio narratives.
